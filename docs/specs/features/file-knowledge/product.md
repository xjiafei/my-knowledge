## 2. 文件管理页（/files）

### 页面目的

提供文件上传、列表浏览、按名称搜索、按类型筛选、按标签筛选、下载和删除的一体化文件管理界面。

### 布局描述

```
┌──────────────────────────────────────────────────────────────┐
│  顶部导航栏（不变）                                             │
├────────────┬─────────────────────────────────────────────────┤
│            │  [页面标题: 文件管理]          [上传文件 按钮]    │
│  侧边栏    ├────────────────────────────────────────────────  │
│  （不变）  │  [搜索框: 按文件名搜索...]  [类型筛选Tab]         │
│            │  全部 | PDF | 文档 | 图片 | 文本 | 压缩包 | 其他  │
│            ├─────────────────────────────────────────────────│
│            │  [标签筛选: el-select 多选 + 已选标签Tag + 清除]  │
│            ├─────────────────────────────────────────────────│
│            │  文件列表（表格/列表视图）                         │
│            │  ┌──────────────────────────────────────────┐   │
│            │  │ 图标 文件名     大小  上传时间  标签  操作 │   │
│            │  │ 📄  report.pdf  2.3MB 2026-03-10  Java ↓🗑│   │
│            │  │ 🖼  photo.png   1.1MB 2026-03-09       ↓🗑│   │
│            │  │ ...                                      │   │
│            │  └──────────────────────────────────────────┘   │
│            ├─────────────────────────────────────────────────│
│            │  [分页器: < 1 2 3 ... >]                         │
└────────────┴─────────────────────────────────────────────────┘
```

### 核心操作

- 点击"上传文件"按钮，弹出上传对话框
- 在搜索框输入文件名关键词（实时过滤，防抖 300ms）
- 点击类型 Tab 切换文件类型筛选
- 在标签筛选区选择/取消标签
- 点击文件行的"下载"图标（↓）下载文件
- 点击文件行的"删除"图标（🗑）弹出确认对话框

### 文件列表列定义

| 列 | 内容 | 说明 |
|----|------|------|
| 图标 | 文件类型图标 | 按 FileCategory 显示不同图标：PDF 红色、文档蓝色、图片绿色、文本灰色、压缩包橙色 |
| 文件名 | 原始文件名（可能 truncate） | hover 显示完整文件名 tooltip；点击文件名展开描述行（若有描述） |
| 文件大小 | 人类可读（如 2.3 MB） | 以 KB/MB 显示，保留一位小数 |
| 上传时间 | 格式：YYYY-MM-DD HH:mm | 按上传时间降序固定排列（P0 不支持切换排序） |
| 标签 | 最多显示 3 个标签 el-tag，超出 "+N" | 点击标签触发标签筛选 |
| 操作 | 下载、删除两个图标按钮 | 图标按钮，hover 显示 tooltip |

### 标签筛选区说明

标签筛选区位于类型 Tab 下方，包含：
- **标签下拉多选（el-select + multiple）**：从现有所有标签中选择，支持搜索
- 已选标签以 `el-tag` 方式展示在下拉框内，可单独点击 × 移除
- "清除筛选"按钮：点击清空已选标签（仅清除标签，不影响类型 Tab 和搜索关键词）
- 文件行中的标签也可点击触发标签筛选（相当于在下拉框中选中该标签）

### 描述展开行（可选）

文件有描述时，点击文件名可展开/收起描述行（el-table 展开行功能）：

```
▼ report.pdf            2.3MB  2026-03-10  Java  ↓🗑
  描述: 2026年第一季度财务分析报告，含收入/支出/利润趋势
```

### 异常场景

| 场景 | 展示 |
|------|------|
| 无文件（初始状态） | 空状态插图 + 文案"暂无文件，点击上传添加知识文件" + 上传按钮 |
| 搜索/筛选无结果 | 空状态文案"未找到相关文件" + 清除筛选按钮 |
| 下载失败 | Toast 提示"下载失败，文件可能已被删除" |
| 删除失败 | Toast 提示"删除失败，请重试" |
| 网络异常 | Toast 提示"网络异常，请检查连接" |
| 磁盘空间不足（上传时 507） | 错误对话框"存储空间不足，请清理磁盘后重试" |

## 3. 上传文件对话框

### 对话框描述

点击"上传文件"按钮，弹出 `el-dialog`（宽度 560px）：

```
┌──────────────────────────────────────────┐
│  上传文件                              [×]│
├──────────────────────────────────────────┤
│                                          │
│  ┌────────────────────────────────────┐  │
│  │    📂 点击选择文件或拖拽至此处      │  │
│  │                                    │  │
│  │  支持：PDF、Word、Excel、PPT、      │  │
│  │  图片、TXT、MD、ZIP（≤ 100MB）     │  │
│  └────────────────────────────────────┘  │
│                                          │
│  已选文件：report.pdf（2.3 MB）          │
│  [进度条 ████████░░ 80%]                 │
│                                          │
│  文件描述（可选）                         │
│  ┌────────────────────────────────────┐  │
│  │                                    │  │
│  └────────────────────────────────────┘  │
│  0/500                                   │
│                                          │
│  标签（可选）                             │
│  [标签多选下拉框 TagSelect]               │
│                                          │
│  [取消]                        [上传文件]│
└──────────────────────────────────────────┘
```

### 上传对话框状态机

```
空闲态（未选文件）
  │ 用户点击选择 / 拖拽文件
  ↓
已选文件态（显示文件名、大小；用户可填描述和标签）
  │ 用户点击"上传文件"
  ↓
上传中态（进度条推进，按钮 loading；P0 不支持取消，用户等待完成或页面刷新中止）
  │ 上传成功
  ↓
成功态（对话框关闭，Toast 提示"上传成功"，列表刷新）

  │ 上传失败（格式/大小/服务器错误）
  ↓
失败态（进度条变红，显示错误文案，可重新选择文件）
```

### 字段说明

| 字段 | 类型 | 必填 | 规则 |
|------|------|------|------|
| 文件 | file input | 是 | 格式见支持列表；大小 ≤ 100MB；客户端和服务端双重校验 |
| 文件描述 | textarea | 否 | 最多 500 字符；超出时字符计数红色提示，不允许提交 |
| 标签 | 多选下拉 | 否 | 从现有 tag 列表选择；最多 20 个 |

### 客户端校验时机

- 选择文件后立即校验格式和大小（不等到提交）
- 描述超 500 字时实时提示，不阻止输入但阻止提交

## 6. 交互流程

### 6.1 主流程：上传文件

```
用户点击"上传文件"
  → 弹出上传对话框
  → 用户选择文件（点击区域 or 拖拽）
    → [客户端] 校验格式：不通过 → 显示格式错误提示，清空已选文件
    → [客户端] 校验大小：不通过 → 显示大小错误提示，清空已选文件
    → 通过 → 显示已选文件名和大小
  → 用户（可选）填写描述
  → 用户（可选）选择标签
  → 用户点击"上传文件"按钮
    → 按钮进入 loading 状态，显示进度条
    → POST /api/files（multipart/form-data）
      → 成功 → 对话框关闭，Toast "上传成功"，文件列表重新加载第 1 页
      → 格式/大小不通过（服务端二次校验）→ 进度条红色，显示服务端错误信息
      → 磁盘不足（507）→ 弹出错误对话框提示清理磁盘
      → 其他错误 → Toast "上传失败，请重试"
```

### 6.2 主流程：下载文件

```
用户点击文件行"下载"图标
  → 发起 GET /api/files/{id}/download
  → 后端流式返回文件内容（Content-Disposition: attachment; filename=原始文件名）
  → 浏览器触发原生下载（进度条由浏览器管理，无额外前端 UI）
  → [若 404] Toast "文件不存在或已被删除"
```

### 6.3 主流程：删除文件

```
用户点击文件行"删除"图标
  → 弹出确认对话框（显示文件名）
  → 用户点击"确认删除"
    → DELETE /api/files/{id}
      → 成功 → Toast "删除成功"，当前页列表刷新
        （若当前页已空且非第 1 页 → 自动跳回第 1 页）
      → 失败 → Toast "删除失败，请重试"
  → 用户点击"取消" → 对话框关闭，不做任何操作
```

### 6.4 筛选组合流程

```
用户进入 /files
  → 默认：全部类型，无标签筛选，搜索框为空，第 1 页
  → 点击类型 Tab（如"PDF"）
    → 更新 typeFilter，重置 page=1，重新请求列表
  → 输入搜索关键词（防抖 300ms）
    → 更新 searchKeyword，重置 page=1，重新请求列表
  → 点击标签（从标签区或文件行的标签点击）
    → 更新 tagFilter，重置 page=1，重新请求列表
  → 各筛选条件叠加（AND 关系）
  → 点击分页器切换页码
    → 仅更新 page，保持其他筛选不变
```

## 8. 与现有系统的集成方案

### 8.1 标签体系复用

- 上传对话框中的 TagSelect 组件直接复用现有 `tagStore.js`，无需新增 API
- 文件管理页的标签筛选使用现有 `/api/tags` 获取标签列表（全量）
- **不修改** TagManagePage、TagCloud、现有笔记标签逻辑

### 8.2 路由集成

在现有 `router/index.js` 末尾追加一条路由：

```javascript
{ path: '/files', name: 'FileList', component: FileListPage }
```

### 8.3 侧边栏集成

在 `AppSidebar.vue` 快捷入口区追加 1 行 router-link，改动范围 ≤ 5 行代码。

### 8.4 数据隔离

- 文件列表与笔记列表完全独立：独立 API、独立 Store、独立页面
- 侧边栏的分类树和标签云点击**不影响**文件管理页（两个页面各自维护筛选状态）
- 标签删除时（现有 TagManagePage 操作）：后端级联删除 file_tag 关联（与 note_tag 同样的级联处理）

### 8.5 影响范围最小化

| 现有模块 | 修改内容 | 修改量 |
|---------|---------|--------|
| `AppSidebar.vue` | 新增"文件管理"链接 | ~3 行 |
| `router/index.js` | 新增 `/files` 路由 | ~1 行 |
| `tag` 后端模块 | `DeleteTag` 时需级联删除 `file_tag`（已有级联机制扩展） | ~2 行 |
| 其他现有模块 | **不修改** | 0 |
